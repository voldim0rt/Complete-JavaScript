DOM Manipulation

Selecting with id

document.getElementById("myId")

Selecting with class

document.getElementsByClassName("myClass")

selecting with tag

document.getElementsByTagName("p")


But there is a better way to access the elements which are the query selectors


Selecting with querySelector

document.querySelector("#myId")
//Returns first element

Selecting with querySelectorAll

document.querySelectorAll(".myClass")
//Returns a Nodelist

Properties

tagname: returns tag for element nodes.

innerText: returns the text content of the element and all its children.

innerHTML: returns the plain text or HTML content in its elements.

innerContent: returns textual content even for hidden elements.
