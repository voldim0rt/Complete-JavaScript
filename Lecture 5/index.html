Callback Functions

A function passed as an argument to the another function.

Map

Create a new array with the results of some operation. The value its callbacks returns
are used to form new array.
arr.map(callbackFun(value, index, array))

Reduce Method

The reduce() method applies a function against an accumulator and each element in the array (from left to right) to reduce it to a single value.
arr.reduce(callbackFun(accumulator, currentValue, currentIndex, array))
